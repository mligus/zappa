# Docker image for Zappa

Docker container to run Zappa based on the [Lambda Docker Image](https://hub.docker.com/r/lambci/lambda/)

Based on [Daniel Whatmuff](https://github.com/danielwhatmuff/zappa) container.

## Configuration
